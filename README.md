# HTML Audio and Video Player

A simple HTML project built as part of my freeCodeCamp coding practice.

## About the Project

This project is a basic multimedia webpage created using HTML. It demonstrates how to embed both video and audio content into a web page using HTML5 media elements.

The page includes:

- A video section explaining the map method
- An audio section with a looping music track
- A simple page structure with headings and sections

## Features

- Embedded video using the `video` element
- Embedded audio using the `audio` element
- Video playback controls
- Audio playback controls
- Looping audio
- Clear content structure using semantic sections

## Built With

- HTML5

## What I Learned

While building this project, I practised:

- Creating a basic HTML document
- Using the `video` element to embed video content
- Using the `audio` element to embed audio content
- Adding playback controls
- Using the `source` tag for video files
- Structuring content with headings and sections
- Building a simple multimedia webpage
